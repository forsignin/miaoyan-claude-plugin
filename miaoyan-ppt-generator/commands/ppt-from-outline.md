从结构化大纲快速生成妙言演示文稿。

适合以下场景:
- 你已经有了演示的思路和大纲
- 需要快速将大纲扩展为完整演示
- 想要基于特定结构生成幻灯片

使用方式:
1. 提供你的演示大纲(可以是文本或文件)
2. 指定演示类型(技术/产品/教学)
3. Skill 会根据大纲自动生成完整的幻灯片内容

大纲格式示例:
```
# 主题: Python 异步编程

## 第一部分: 基础概念
- 什么是异步编程
- 为什么需要异步
- 同步 vs 异步

## 第二部分: asyncio 库
- 基本用法
- async/await 语法
- 代码示例

## 第三部分: 实战应用
- Web 爬虫
- API 调用
- 最佳实践
```

输出: 完整的妙言 PPT,包含封面、章节页、内容页和 Q&A
